Portfolio

Quantum Coasters

Nashville Software School front end capstone project. As a roller coaster fan myself, my goal was to build an app for roller coaster enthusiast to track their coaster credits. Quantum also encompases a thorough, usable, crowd sourced API/ database of roller coaster information and locations, as well as being an interactive application where users can record their credits, see a live leaderboard, contribute to the app's database, and message other users in a chat / forum. Please see ReadMe on Github for more details as well as list of technology used.

Fōlô React App

Application built to demonstrate expertise in implementing full CRUD functionality. Fōlô (pronounced follow) has features such as login, task tracking, a live chat/ messaging component, ability to add and follow friends, search and save news articles, search events near the user, and perform full CRUD on all of these features.

NutShell

Goal: to design a basic single page application using vanilla JS and no library or framework, that utilizes all aspects of CRUD. App is built with a top priority to be readable and user friendly, and includes (4) sections: Events, Tasks, Friends, and Messages. Follow link to Github to see the project, and the README for more details! Nutshell is deployed using Netlify. To see it in production environment, click here

Keahua Arboretum

It is written with modular, class based code using techniques such as Python classes, object oriented design, interfaces (python multiple inheritance), thorough duck typing, many of Python's data structures, saving to a txt file, and printing out to the command line.
