Recent projects I have worked on.

Quantum Coasters

Nashville Software School front end capstone project. Goal was to build an app for roller coaster enthusiast to track their coaster credits. Built a thorough, usable, crowd sourced API/ database of roller coaster information and locations, as well as created an interactive application where users can record their credits, see a live leaderboard, contribute to the app's database, and message other users in a chat / forum.

Icon

Fōlô React App

Application built to demonstrate expertise in implementing full CRUD functionality. Fōlô (pronounced follow) has features such as login, task tracking, a live chat/ messaging component, ability to add and follow friends, search and save news articles, search events near the user, and perform full CRUD on all of these features.

Icon

NutShell

Goal: to design a basic single page application using no library or framework, that utilizes all aspects of CRUD. We were given instructions and acceptance criteria from our "product owner" , and to implement the functionality to be readable and user friendly. App includes (4) sections: events, tasks, friends, and messages. Follow link to Github to see project, and the README for more details!

Icon

Welcome To Nashville

The goal of this project was to build a simple app that utilizes API information and prints it to an itinerary. We used APIs from Ticketmaster, Nashville Government, and Zomato. This was our second group project at Nashville Software School, and we used skills and concepts including Functions, Persistent data storage, Github, CSS, Building DOM components, Handling user events, Modular code, Semantic HTML, and Valid HTML5.
