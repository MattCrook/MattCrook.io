Portfolio

Quantum Coasters

As a roller coaster enthusiast, my goal was to build a place for roller coaster fans and other enthusiasts to track their coaster credits. Quantum Coasters also encompasses a thorough, usable, crowd sourced API/ database of roller coaster information and locations, as well as being an interactive application where users can record their credits, see a live leaderboard, contribute to the app's database by adding roller coasters, and message other users in a chat / forum. This is a full-stack application with a React frontend and Django REST API on the backend. Auth0 is used for authentication and authorization, and is fully integrated into both the front and back ends. Please see ReadMe on Github for more details as well as list of technology used. Back end repo here.

Quantum Management

Sticking with the roller coaster and amusement park theme, Quantum Management is a full-stack mostly server-side rendered application that is built using Python/ Django, and also includes a Django REST API using Django Rest Framework for various client side rendering cases where an endpoint needed to be exposed. Quantum Management is a theme park management and business analysis management application, who's purpose is to track employees at the park, where they work, allow for addition, removal, or transition to a different role. In addition, Quantum Management provides a dashboard of analytical business intelligence information for educated business decisions, reporting, forecasting, and much more customizable queries to derive financial and income tracking data. All of which are displayed using data visualization tools for easy readability and analysis. For a more detailed ReadMe, click the Github link below!

Iris Station - Eccommerce Application

Iris Station is a full stack eccommerce application utilizing a Django Rest API backend, and React frontend. It provides all of the functionality you would see and expect in an online shopping site such as Amazon. In addition, the application has 100% test coverage using pytest. To see the React front end repo and a more detailed description on Github, click here. To view the back end Django API click here. This application was a group effort, and includes a multitude of features, too many to list! See the Github repo for full list of all features, as well as the tech stack used.

Fōlô React App

Single page application built with React to demonstrate expertise in implementing full CRUD functionality. Fōlô (pronounced follow) has features such as login, task tracking, a live chat/ messaging component, ability to add and follow friends, search and save news articles, search events near the user, and perform full CRUD on all of these features.

Calendar In Javascript

Calendar written in vanilla Javascript. See the app here.

    Features of the calendar are:

  • Render the months onto a dropdown.
  • Render the years onto a dropdown.
  • Render the days for the currently selected month/year combination.
  • Redraw the calendar when the months/ years are changed
  • Changing the months and/or year will redraw the calendar to match the correct days per month and the correct offset in the week.

Python Sockets and Multi Threading

This is a personal project and fun experiment that was inspired by a YouTube video I watched on multi-threading in Python. I decided to take the idea and run with it, creating this small command line project. This program will allow anyone running it to chat with other users on the same local IP address, also running the program on the command line. It will also show how many connections (i.e other people using the program) there are.
